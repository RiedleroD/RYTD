# **RYTD**
_or Riedler YouTube Downloader_

-----

RYTD is an automated music downloader, supporting [a few websites](http://ytdl-org.github.io/youtube-dl/supportedsites.html). 

Aviable command line arguments:

- `--configure`|`-c`
- `--help`|`-h`|`?`
- `--overwrite`|`-o`
- `--verbose `|`-v`
- `--warnings`|`-w`
